# ☕ Starbucks Sales Analytics Dashboard | Power BI

An interactive **Power BI dashboard** developed to analyze Starbucks sales data, monitor key business KPIs, and provide actionable insights through dynamic visualizations.

---

## 📌 Overview

This project provides a comprehensive analysis of Starbucks sales performance across different stores, products, and time periods. The dashboard enables users to monitor sales trends, customer purchasing behavior, and product performance using interactive reports and filters.

---

## 🚀 Features

- 📈 Interactive Sales Dashboard
- 💰 Sales & Revenue Analysis
- 🛍️ Product Category Performance
- ☕ Best-Selling Products
- 🏪 Store-wise Sales Analysis
- 📅 Daily & Monthly Sales Trends
- 📊 KPI Cards & Dynamic Visualizations
- 🎯 Interactive Filters & Slicers

---

## 📊 Key KPIs

- **Total Sales**
- **Total Orders**
- **Total Quantity Sold**
- **Average Order Value**
- **Average Sales per Transaction**

---

## 🛠️ Technologies Used

- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization
